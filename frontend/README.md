# SAM — Frontend

Frontend React do projeto SAM.

## Setup inicial (a executar quando começarem o desenvolvimento)

```bash
npm create vite@latest . -- --template react-ts
npm install
npm run dev
```

Sugestões de bibliotecas a avaliar conforme o projeto avançar:
- `react-router-dom` — rotas
- `axios` ou `fetch` nativo — comunicação com o backend
- `tailwindcss` — estilização mobile-first
